## 💻 Sobre o projeto

Apresento um projeto de calculadora de Índice de Massa Corporal (IMC) desenvolvido em React Native. Este aplicativo proporciona uma maneira fácil e rápida de os usuários calcularem seu IMC, promovendo a conscientização sobre saúde e bem-estar.
